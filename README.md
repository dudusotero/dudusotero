### What I'm building

**[Pixelore UI](https://github.com/dudusotero/pixelore)** is an 8-bit React design system
for browser games. Mainstream design systems target dashboards; this one targets the thing
they don't. 18 Radix-based primitives, dual ESM+CJS distribution, a Tailwind v3/v4 preset,
and WCAG 2.2 AA throughout. Published on npm as `@pixelore/react`.

It is proven by [a complete turn-based RPG](https://pixelore-rpg.vercel.app) built on top
of it rather than by a gallery of isolated examples. Every component had to survive a real
app.

[Docs](https://pixelore-ui.vercel.app) · [Play the RPG](https://pixelore-rpg.vercel.app) · [How it was built](https://www.dudusotero.dev/work/pixelore-ui)
